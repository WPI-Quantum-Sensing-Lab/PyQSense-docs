class PyQSense.aux_control.SCPI_RF_Generator(address, model_name, err_check_method='scpi', **kwargs)

Microwave of RF source instrument.

A microwave or RF source, supporting the base functionality of the SCPI “RF & Microwave Source” instrument class.

enable_output()

Open the output terminals.

disable_output()

Close the output terminals.

Per SCPI specification, this means that “the terminals are at maximum isolation from the signal”.

set_amplitude(amplitude)

Set the amplitude of the signal (in dBm).

Per specification, the command :SOURce:POWer:LEVel:IMMediate:AMPLitude <numeric-value> is implied by :SOURce:POWer <numeric-value>. Furthermore, the units on the value are set to the default, which depends on the device. The :UNIT:POWer DBM command will set the default units for the power subsystem to decibel-milliwatts. However, some devices which are not fully SCPI-compliant (TPI-1001-B) do not implement the unit subsystem. An alternative method of specifying the units is by appending the units to the end of a command.

Parameters:

amplitude (float) – The amplitude to set, in dBm.

set_frequency(freq)

Set the frequency of the signal (in Hz).

The standard does not support any unit other than Hz. However, devices may implement other default units.

Parameters:

freq (float) – Frequency to set, in Hz.

class PyQSense.aux_control.TPI_RF_Generator(address, model_name, err_check_method='generic', baud_rate=3000000)

Control a TPI-1001-B microwave generator.

This class implements two modes of communication with the TPI: SCPI and a vendor-specific packet protocol. The device only supports a limited subset of SCPI commands.

Note

The SCPI control mode is available from firmware version 1.074 or newer, and must be enabled in TPI-Link, the companion software.

address

VISA address of the TPI device.

Type:

str

_instrument

Internal object, representing the instrument as a VISA resource.

Type:

pyvisa.resources.Resource

__init__(address, model_name, err_check_method='generic', baud_rate=3000000)

Initalize the connection to the RF generator.

Parameters:

  • address (str) – VISA address of the TPI device.

  • model_name (str) – not yet implemented.

Raises:

RuntimeWarning – There was an issue in enabling “user control” mode, a requirement for the packet-based communication protocol.

set_amplitude(rf_amplitude)

Sets the power of the output signal.

The TPI’s resolution is 1 dBm. If a non-integer value is passed as rf_amplitude, it will be converted to an integer by rounding down.

Parameters:

rf_amplitude (int) – The output power to set, in dBm.

Raises:

ValueError – The amplitude has been requested to be a value less than -90 dBm. The manufacturer discourages such low values.

set_frequency(freq)

Sets the frequency of the output signal in Hz.

The frequency resolution is up to the third decimal place, in MHz.

Parameters:

freq (float) – the frequency to set, in Hz.
